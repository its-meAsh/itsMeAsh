Guide Of Projects

Memory game
Interactive
Responsive
Game
This project is a game whose backbone is time, and therefore system performance would surely affect the user's experience. This game is similar to Google's memory game. When you start the game with level 1, 2 cards in an order will raise up with a red border around them, after a second, the cards will come to rest. Now it's your turn to click on those cards which were raised and in the same order in which they were. Once you click on all the cards, you will be promoted to the next level however if you fail to do so, you will only loose a level. Once you reach level 7, it is the last level.
Rock Paper Scissor
Interactive
Responsive
Game
This project is another game partially based on AI. The JavaScript made bot records your moves, analyzes them and thus tries to predict what your next move can be and thus defend itself and get a point. The bar at the top right of the page (only for PCs and Laptops) is a scripted bot which texts relating to the scoreboard. It has not been designed very specifically and thus some of it's texts can be weird to the situation it was and what it is now.
Convertor
Interactive
Responsive
Tool
This project is a Text to ASCII, ASCII to Text, Text to Hexadecimal, Hexadecimal to Text, ASCII to Hexadecimal and Hexadecimal to ASCII convertor all simultaneously. As you type in any of the text boxes, if it is valid, then the changes can be seen in the other two and thus you can convert them without needing to specify to what you want tot convert them to as all the options are right on your screen.

This project is a tool where you toss an unbiased coin to get an outcome and along with that get some statistical data and probability of what the next occurence can be and by how much percentage it can be.
Number game
Interactive
Responsive
Game
This project is similar to the 1st project, the only difference is that in this, you need to remember numbers and then once the displaying of number stops, you need to enter your answer in the input box. It has infinite levels and the final checkpoint for this game is level 10 after which you can play it further as well. If you couldn't remember the number correctly and enter a wrong number, you will only loose a level.
