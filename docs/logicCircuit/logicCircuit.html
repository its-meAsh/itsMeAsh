Logic Circuits
What is a logic gate?
A logic gate is a device that performs a Boolean function, a logical operation performed on one or more binary inputs that produces a single binary output.
What is a truth table?
A truth table is a mathematical table used in logic - in connection with Boolean algebra, Boolean functions and propositional calculus. It sets out the functional values of logical expressions on each of their functional arguments, i.e. for each combination of values taken by their logical variables.
Types of logic gates
OR Gate
An OR gate returns true if either or all the inputs are true else returns false. This gate is also known as inclusive OR.
class LogicGate {
  or(input1, input2) {
    if (input1 || input2) {
      return true;
    }
    else {
      return false;
    }
  }
}
AND Gate
An AND gate returns true only if all the inputs are true else returns false.
class LogicGate {
  and(input1, input2) {
    if (input1 && input2) {
      return true;
    }
    else {
      return false;
    }
  }
}
XOR Gate
An XOR gate returns true only if any one of the inputs is true else returns false. This gate is known as exclusive OR.
class LogicGate {
  xor(input1, input2) {
    if ((input1 && !input2) || (!input1 && input2)) {
      return true;
    }
    else {
      return false;
    }
  }
}
NOT Gate
A NOT gate takes only 1 input and returns true when the input is false and returns false when the input is true i.e. opposite of the input.
class LogicGate {
  not(input1) {
    return !input1;
  }
}
Buffer Gate
A Buffer gate takes only 1 input and returns the same input as the output. It is used in Digital Electronics to create delays.
class LogicGate {
  buffer(input1) {
    return input1;
  }
}
NAND Gate
A NAND gate is an AND gate followed by a NOT gate. It returns true when either of the inputs are false else returns false.
class LogicGate {
  nand(input1, input2) {
    if (input1 && input2) {
      return !true;
    }
    else {
      return !false;
    }
  }
}
NOR Gate
A NOR gate is an OR gate followed by a NOT gate. It returns true when all the inputs are false else returns false.
class LogicGate {
  nor(input1, input2) {
    if (input1 || input2) {
      return !true;
    }
    else {
      return !false;
    }
  }
}
XNOR Gate
An XNOR gate is an XOR gate followed by a NOT gate. It returns true if all the inputs are true or false else returns false.
class LogicGate {
  xnor(input1, input2) {
    if ((input1 && !input2) || (!input1 && input2)) {
      return !true;
    }
    else {
      return !false;
    }
  }
}
