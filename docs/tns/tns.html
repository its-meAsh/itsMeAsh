Transition and Animation
Transition
Transitions in CSS are set using the transition property. Transitions are single keyframed and thus only a specific special effect can be obtained. Transitions help to add frames to one or many property/properties of an element which change(s) on a certain occasion such as color, font-size, transform etc.

Animation
Animations in CSS are set using the animation property. Animation are multi-keyframed and thus many special effect can be obtained. Animation help to add frames to one or many property/properties of an element which change(s) either for a specific duration or forever. The good thing about animations is that one can create a sequence of all the happenings in animations unlike transition where all the effects happen at once.

Transition and Animation in the homepage
  • Starting from the navigation bar, each option at the navigation bar has a transition to rotate 20 degrees when hovered.
  • Followed by that is the first row on which hovering results in the interchange of their positions.
  • The third effect is in the Skills section where hovering each skill moves it to right and simultaneously decreases its opacity from 1 to 0
  • The fourth effect is my favorite one and I call it as the Frame fall animation. It is controlled by JavaScript. On the first and second hover, the hovered frame swings 20 degrees making it look like it is hanging on just one nail. Doing this twice results in the same animation but when it is hovered the third time, it swings completely and fall off. Ofcourse the frames again get back to their respective places.
  • Following this is the 5th transition to scale up each project tile as each of them is hovered.
  • In the Doc section, a basic transition is to move each Doc box to the right, decrease its opacity, then make it come from the left with an increasing opacity.
  • At the bottom lie the contact options with a border color changing transition on each when hovered.
  • A pattern to notice is that every word which spells Ash is colored #30D2C5 and hovering it results in changing of its color to white.
  • The last animation is in every word refering to India has a color changing animation to the three colors of the Indian flag
